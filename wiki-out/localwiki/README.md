# LocalWiki 프로젝트 위키

LocalWiki는 GitHub, GitLab, BitBucket 저장소를 분석해 코드 구조, 문서, 다이어그램, 대화형 위키를 자동 생성하는 풀스택 문서화 플랫폼입니다.
